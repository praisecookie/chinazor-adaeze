# Adaeze Chinazor Chukwuemeka | Professional Portfolio

A modern, responsive portfolio website showcasing over 12 years of excellence in Early Education, Event Planning, and Culinary Arts.

## Features

- **Color-Coded Expertise Hub:** Distinct, easy-to-navigate sections for Education (VRC Academy, TOM), Events, and Culinary services.
- **Smart Contact Form:** Integrated directly with Google Sheets and Apps Script to trigger automated HTML email notifications to both the site owner and the inquirer.
- **Fully Responsive:** Optimized for all screen sizes with a custom mobile navigation menu.

## Tech Stack

- Next.js 15 (App Router)
- Tailwind CSS v4
- TypeScript
- React Icons & Lucide React

## Running Locally

1. Clone the repository:

   ```bash
   git clone [https://github.com/YOUR_USERNAME/chinazor-adaeze.git](https://github.com/YOUR_USERNAME/chinazor-adaeze.git)
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open http://localhost:3000 in your browser.
